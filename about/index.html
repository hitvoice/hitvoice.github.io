Xinya: Chinese Lyrics Generation
A Chinese lyrics generation system based on generative adversarial network. Autoencoder is used to encode a lyric line into a latent code, and the generator tries to generate a "realistic" latent code based on the current context which will be decoded into a new lyric line. A n-gram language model helps to improve the generated result and control the length of the generated line. The autoencoder is pretrained optimizing the cross entropy and then the GAN is trained under the WGAN objective. The qualily of generation is way better than previous maximum likelyhood methods (like seq2seq).

Intelligent Case Management System
A intelligent case management system for the Government Hotline Service Center of Nanjing. It integrates GIS and NLP techniques into the ordinary case management system, which enables the telephone operators to quickly mark associated locations on the map, assign cases following case assignment recommendations and enable the governors to know what people in this city are complaining about through visualizations of text clustering results and hard-to-deal-with cases through outlier analysis.

Graphic Design to 3D Printing
Convert 2D graphic designs to 3D-printable models. The demo above illustrates converting a SVG file to a STL model, by sampling vertices along Bezier paths, performing constrained Delaunay triangulation, and converting a 2d mesh to 3D. (demo code)

Virtual Fitting Room
A virtual fitting system based on Kinect and 3D human models. A picture of clothes with similar posture is queried using ANN-search, and further streched by a SCAPE 3D human model which is estimated from input RGBD frames.

NJU Exchange Notifiers
This is a personal project, a mail notifier for notifications from the Exchange Center of Nanjing University. The Exchange Center updates international exchange student programs on its website, applications for which usually close in a few days. It's simple in techniques -- just a crawler and a mail agent -- but has helped a lot of real users to get notified in time!
