BM-25: Term frequency weight

BM25(w,D)=1+k1+k/(c(w)1+b(|D|−|D¯|)/|D¯|)

  • c(w): word count in document D
  • |D|: document length
  • |D¯|: average document length
  • parameter k∈[0,∞): set upper bound to k+1
  • parameter b∈[0,1]: control length normalization
